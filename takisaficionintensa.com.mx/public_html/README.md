# Takis Afición Intensa - Página de Cuenta Regresiva

## Descripción
Página de cuenta regresiva para el lanzamiento de Takis Afición Intensa el 16 de febrero de 2026.

## Características
- ✅ Diseño responsivo (desktop, tablet, mobile)
- ✅ Contador regresivo en tiempo real
- ✅ Tipografía Veneer personalizada
- ✅ Animaciones suaves y efectos visuales
- ✅ Fondo de textura de Takis
- ✅ Overlay de imagen con efecto morado
- ✅ Logo de Takis animado

## Estructura de Archivos
```
takisaficionintensa.com.mx/public_html/
├── index.html          # Página principal
├── styles.css          # Estilos y diseño responsivo
├── script.js           # Lógica del contador regresivo
├── logo-takis.png      # Logo de Takis
├── takis-texture.jpg   # Textura de fondo (Takis rojos)
├── background-people.jpg # Imagen de fondo (personas)
└── fonts/
    └── Veneer.otf      # Tipografía personalizada
```

## Fecha del Contador
El contador está configurado para el **16 de febrero de 2026 a las 00:00:00**.

## Tecnologías Utilizadas
- HTML5
- CSS3 (con animaciones y efectos glassmorphism)
- JavaScript vanilla
- Tipografía Veneer

## Diseño Responsivo
El sitio se adapta a diferentes tamaños de pantalla:
- **Desktop**: Vista completa con todos los elementos
- **Tablet** (768px): Ajustes de tamaño y espaciado
- **Mobile** (480px): Layout optimizado para pantallas pequeñas

## Colores Principales
- Morado principal: `#6B2D9E` - `#8B3FC4`
- Amarillo dorado: `#FFD700`
- Naranja: `#FF6B00`
- Blanco: `#FFFFFF`

## Instalación
1. Subir todos los archivos al servidor web
2. Asegurarse de que la estructura de carpetas se mantenga
3. El sitio estará disponible en `takisaficionintensa.com.mx`

## Notas
- El contador se actualiza cada segundo
- Las animaciones incluyen efectos de hover y transiciones suaves
- El diseño utiliza efectos de glassmorphism para los números del contador
- La tipografía Veneer se carga localmente desde la carpeta `fonts/`
